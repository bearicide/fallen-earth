# Fallen Earth

An abstract post-modern interactive 16:9 web background / toy.

It is meant to feel like a ruined cinematic world that reacts to whatever you throw at it.

## Interactions

- Move mouse / touch: parallax and light tracking
- Click / tap: burst
- Hold + drag: trails, bolts, sparks
- Wheel: zoom pulse and color shift
- Keyboard: every key does something
- Paste text: rune burst
- Drop files: meteor storm
- Right-click: alternate burst

## Run

Open `index.html` in any modern browser.
