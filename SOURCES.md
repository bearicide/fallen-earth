# Sources

Project source/reference notes for Fallen Earth.

## Repository

- GitHub repo: `bearicide/fallen-earth`
- GitHub Pages: `https://bearicide.github.io/fallen-earth/`

## Core direction

- 16:9 cinematic ruined-world background
- Slightly abstract / somewhat post-modern visual treatment
- Interactive HTML page with little to no visible UI
- The world itself acts as the interface
- Mouse, touch, keyboard, paste, drag/drop, wheel, and MIDI input all trigger visual/audio reactions

## Current feature sources

- Procedural CSS/canvas environment
- Procedural Web Audio sound engine
- Web MIDI input activation through the central obelisk/tower interaction
- One-time obelisk MIDI activation that fades out after use

## Interaction map

- Move: parallax and light tracking
- Click/tap: impact burst and sound wake
- Hold/drag: trails, sparks, and bolts
- Wheel: zoom/color pulse
- Keyboard: every key triggers sound/visual response
- Paste: rune burst
- Drop files: meteor storm
- Right-click: alternate burst
- MIDI notes: trigger position-based bursts and tones
- MIDI CC: affects hue, zoom, charge, and tone
