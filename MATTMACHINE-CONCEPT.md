# MATTMACHINE - Void → Forest

A single interactive journey.

## VOID
- Infinite stars
- Particle trails
- Draw constellations
- Gravity wells on click
- Ambient drone

Transition:
The void cracks and grows roots.
Stars become fireflies.

## FOREST
- Dream forest
- Synth mushrooms
- Floating lights
- Hidden MIDI shrine
- XY pond controller

Core philosophy:
Touch stuff. Something happens.
No dead buttons.
No boring menus.

The site should feel like a toy, instrument, and dreamspace at the same time.
